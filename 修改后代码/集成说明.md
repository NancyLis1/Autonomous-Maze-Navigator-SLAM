- **引脚：** PA0（默认配置）
- **连接方式：** 低电平有效
- **电路：** 按钮一端接PA0，另一端接GND

### 4. 常见问题

**Q: 编译报错找不到头文件**
A: 检查MDK工程是否添加了4个新的.c文件

**Q: 急停无效**
A: 检查GPIO引脚配置是否正确

**Q: 位姿控制震荡**
A: 降低PID参数：
```c
g_pose_config.linear_kp = 150.0f;  // 原200
g_pose_config.angular_kp = 100.0f; // 原150
```

**Q: 里程计漂移严重**
A: 校准轮径和轮距：
```c
g_odom_config.wheel_radius = 0.035f;  // 用卡尺测量
g_odom_config.wheel_base = 0.18f;     // 测量实际轮距
```
